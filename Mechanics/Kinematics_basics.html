Basic kinematics

Kinematics questions are questions about the motion of objects. The objects are moving with constant acceleration. If the objects are moving with "smoothly" changing acceleration then use calculus techniques; see the following notes.

When approaching kinematics problems, success depends on following a systematic approach. This guide outlines the essential steps and considerations for solving kinematics questions effectively.

Why do we talk about particles?

In mechanics, we often model objects as particles – point masses with no size or shape. A particle has a position expressed either as a 2D or 3D coordinate such as (3, -4) or (7, -1, -3). Position can also be expressed as the vector p or the vector q, but to calculate with these, values need to be given for the position components of the position vector. This simplification is valid when:

  • The size of the object is negligible compared to the distances involved in the motion, in other words a particle has position/location but no length or breadth.
  • We can ignore rotational effects and focus only on translational motion
  • All parts of the object move together as one unit

Maximum height

\[H = \frac{u^2}{2g}\]

An object is thrown vertically upwards from a surface with an initial speed of \(u\). Determine the maximum height the object reaches before it falls back to the ground.

Learn More →

Solution:

Given information:

• Initial velocity: \(u\) m/s (upwards)

• Final velocity at maximum height: \(v = 0\) m/s

• Acceleration: \(a = -g\) (downwards, so negative)

• Displacement to maximum height: \(s = H\) (what we need to find)

Analysis:

At the maximum height, the object momentarily stops before falling back down, so the velocity at maximum height is zero.

Finding maximum height:

Using \(v^2 = u^2 + 2as\):

\[0^2 = u^2 + 2(-g)H\]

\[0 = u^2 - 2gH\]

\[2gH = u^2\]

\[H = \frac{u^2}{2g}\]

Answer:

The maximum height reached is \(H = \frac{u^2}{2g}\)

Note:

This formula shows that the maximum height is proportional to the square of the initial velocity. If you double the initial speed, the maximum height increases by a factor of four.

Time of flight

\[T = \frac{2u}{g}\]

An object is thrown vertically upwards from the ground with an initial speed of \(u\). Determine the time the object takes to fall back to the ground.

Learn More →

Solution:

Given information:

• Initial velocity: \(u\) m/s (upwards)

• Acceleration: \(a = -g\) (downwards, so negative)

• Displacement when back at ground: \(s = 0\) m

• Time to return to ground: \(t = T\) (what we need to find)

Analysis:

When the object returns to the ground, it is back at its starting position, so the total displacement is zero.

Finding time of flight:

Using \(s = ut + \frac{1}{2}at^2\):

\[0 = uT + \frac{1}{2}(-g)T^2\]

\[0 = uT - \frac{1}{2}gT^2\]

Factor out \(T\):

\[0 = T\left(u - \frac{1}{2}gT\right)\]

This gives two solutions: \(T = 0\) (initial launch) or \(u - \frac{1}{2}gT = 0\)

Solving for the non-zero solution:

\[u - \frac{1}{2}gT = 0\]

\[\frac{1}{2}gT = u\]

\[T = \frac{2u}{g}\]

Answer:

The time of flight is \(T = \frac{2u}{g}\)

Note:

This is exactly twice the time it takes to reach maximum height. The object takes the same time going up as it does coming down.

Total distance travelled

\[H = \frac{u^2}{g}\]

An object is thrown vertically upwards from the ground with an initial speed of \(u\). Determine the total distance the object travels from initial projection to when it falls back to the ground.

Learn More →

Solution:

Given information:

• Initial velocity: \(u\) m/s (upwards)

• Acceleration: \(a = -g\) (downwards, so negative)

• Object starts and ends at ground level

Analysis:

Distance is different from displacement. The object:

1. Travels upward from ground to maximum height

2. Travels downward from maximum height back to ground

Step 1: Find maximum height

At maximum height, velocity \(v = 0\)

Using \(v^2 = u^2 + 2as\):

\[0^2 = u^2 + 2(-g)H\]

\[0 = u^2 - 2gH\]

\[H = \frac{u^2}{2g}\]

Step 2: Calculate total distance

Distance up = \(H = \frac{u^2}{2g}\)

Distance down = \(H = \frac{u^2}{2g}\)

Total distance = Distance up + Distance down

\[\text{Total distance} = \frac{u^2}{2g} + \frac{u^2}{2g} = \frac{2u^2}{2g} = \frac{u^2}{g}\]

Answer:

The total distance travelled is \(\frac{u^2}{g}\)

Important note:

The total displacement is zero (starts and ends at the same point), but the total distance is \(\frac{u^2}{g}\) because distance doesn't consider direction.

Understanding Motion

Motion is the change in position of an object over time. When an object moves, it changes its location relative to a reference point or coordinate system. Motion can be described in terms of how far something moves, how fast it moves, and whether its speed is changing.

Distance is the total length of the path traveled by an object, regardless of direction. It is always a positive value and is measured in units such as meters (m), kilometers (km), or miles. Distance tells us "how much ground was covered" during the motion.

Speed is the rate at which distance is covered over time. It tells us how fast an object is moving and is calculated as distance divided by time. Speed is measured in units such as meters per second (m/s) or kilometers per hour (km/h). Like distance, speed is always positive or zero.

Acceleration is the rate at which speed changes over time. When an object speeds up, slows down, or changes direction, it is accelerating. Acceleration is measured in units such as meters per second squared (m/s²). Positive acceleration means the object is speeding up, while negative acceleration (sometimes called deceleration) means it is slowing down.

Directed motion

In mechanics or kinematic questions most quantities are 'directed' - or have a direction. When you move a certain distance, you also move in a direction: up, down, left, right, North, South etc. The effect of your movement results from both how far you move from your start point and in which direction you moved. In order to determine your direction of movement you need some axes - more on that later.

Movement expressed as distance plus direction is called displacement. Speed plus direction is called velocity. Acceleration plus direction is, well, acceleration. Time does not have a direction in the sense of up, down, left, right but we can think of a positive time as being something happening in the future and a negative time as being something happening in the past.

Indicators of a constant acceleration or suvat question

Recognizing when a problem requires constant acceleration, or suvat, methods is the crucial first step in solving it. Here are the key indicators to look for:

Keywords: Look for words like "constant acceleration", "starts from rest", "the vehicle passes the start with velocity u ms⁻¹", etc.

Direction of motion: In simple (1D) problems the object such as a car or a ball is travelling in a straight line such as falling vertically or travelling along a horizontal surface. A particle travelling along a flat inclined plane is travelling in 'one' direction parallel to the plane but this requires resolving velocities (examples later).

Note: Problems may not explicitly state "constant acceleration" or "kinematics" but the context will indicate it (e.g., a vehicle accelerating uniformly, an object falling freely, or a ball rolling down a slope).

Why diagrams are essential

Drawing a clear diagram is one of the most important steps in solving projectile motion problems. A good diagram helps you visualize the problem, identify what you know, and determine what you need to find.

Coordinate system: Always establish a clear coordinate system. The direction of motion can be horizontal or vertical and we could label this as the x-axis. If this problem or situation has one axis then the object can only move in a straight line, but in either direction. If the object is a projectile, moving horizontally and vertically at any moment, the x-axis could be horizontal and the y-axis vertical, with the origin at a convenient location (often the launch point). With two axis or direction of motion, the object can move anywhere on a flat surface, a plane. Or the object could be moving along the direction of horizontal ground and moving vertically (up and down) at the same time. If the object is moving along a flat plane or slope, the x-axis is normally parallel to the greatest slope and the y-axis is perpendicular, or at right-angles, to the slope.

Initial position: Mark the starting position of the projectile clearly on your diagram. All displacements are measured from this position. If an object returns to this point after mving, its displacement is zero at that point, no matter how far the object has moved.

Velocity component(s): Draw and label the initial velocity, u, and its initial direction (or horizontal and vertical components \(u_{x}\) (\(v_{0x}\)) and \(u_{y}\) (\(v_{0y}\)) , if a projectile).

Key points: Mark important points such as the maximum height, landing position, or any target locations mentioned in the problem.

Known and unknown quantities: Label all known values and identify what you need to find.

Tip: A well-drawn diagram can often reveal the solution method before you even write an equation!

Reference axes

Axes must have a start point, or origin. One axis must point in a straight line in one direction, the second axis (if required) must point in one of the directions perpendicular (at right angles) to the first axis. All directed quantities must have their direction expressed in relation to the axes.

Notation

If a problem is about (uniform acceleration) motion in a straight line (it is one-dimensional), then we can use the following variables to express and solve our problem:

  • \(s\) is displacement from origin
  • \(u\) is initial velocity
  • \(v\) is final velocity
  • \(a\) is (constant) acceleration
  • \(t\) is time (a non directed quantity)

If a problem is about (uniform acceleration) motion in a plane (it is two-dimensional), then we can use the following variables to express and solve our problem:

  • \(s_x\) and \(s_y\) are displacement in the x and y direction
  • \(u_x\) and \(u_y\) are initial velocity in the x and y direction
  • \(v_x\) and \(v_y\) are final velocity in the x and y direction
  • \(a_x\) and \(a_y\) are acceleration in the x and y direction

Constant acceleration equations

The five standard constant acceleration equations (also known as suvat equations) relate displacement, initial velocity, final velocity, acceleration, and time:

  • \(v = u + at\)
  • \(s = ut + \frac{1}{2}at^2\)
  • \(s = vt - \frac{1}{2}at^2\)
  • \(s = \frac{1}{2}(u+v)t\)
  • \(v^2 = u^2 + 2as\)

Where: s = displacement, u = initial velocity, v = final velocity, a = acceleration, t = time

In a kinematics question you will be given 3 variable values, often expressed in words like 'initially at rest' and be asked to determine the value of one other variable. So the question will be about 4 variables, three known values, one to be determined. Find the equation that relates these four variables and apply it to the values.

Essential steps for solving 1D constant acceleration problems:

  1. Read the question carefully: Identify what the problem is asking for and what information is given. Look for keywords like "initially at rest" (u=0), "dropped" (u=0), or "comes to rest" (v=0).
  2. Draw a diagram: Sketch the motion showing the start and end positions. Mark the positive direction clearly (usually upward is positive for vertical motion, or the direction of initial motion for horizontal).
  3. List the known quantities: Write down the values for s, u, v, a, and t that are given in the problem. Remember: s=0 at the starting position, t=0 at the start time.
  4. Identify what you need to find: Clearly state which variable(s) you need to calculate.
  5. Choose the appropriate suvat equation: Select the equation that contains the three known quantities and the one unknown you're solving for.
  6. Substitute and solve: Plug in the known values with correct signs (positive or negative based on direction) and solve for the unknown. Show your working clearly.
  7. Check your answer: Does the sign make sense? Is the magnitude reasonable? Does it have the correct units?

Example problems

Displacement

\[s = ut + \frac{1}{2}at^2\]

A particle travels with uniform acceleration along a horizontal straight line, initially at rest over a distance of 20 metres that takes 8 seconds. Find its acceleration and velocity after 10 seconds.

Learn More →

Solution:

Given information:

• Initial velocity: \(u = 0\) m/s (initially at rest)

• Displacement: \(s = 20\) m

• Time for this displacement: \(t = 8\) s

Part 1: Find acceleration

Using \(s = ut + \frac{1}{2}at^2\):

\[20 = 0(8) + \frac{1}{2}a(8)^2\]

\[20 = \frac{1}{2}a(64)\]

\[20 = 32a\]

\[a = \frac{20}{32} = 0.625\text{ m/s}^2\]

Part 2: Find velocity after 10 seconds

Using \(v = u + at\):

\[v = 0 + 0.625(10)\]

\[v = 6.25\text{ m/s}\]

Answer:

• Acceleration: \(a = 0.625\text{ m/s}^2\)

• Velocity after 10 seconds: \(v = 6.25\text{ m/s}\)

Time for travel and acceleration

\[s = \frac{1}{2}(u+v)t\]

An object accelerates uniformly from 2 ms\(^{-1}\) to 10 ms\(^{-1}\) along a straight line. During this acceleration the object travels 48m. Calculate: a) the time taken for the travel b) the acceleration of the object.

Learn More →

Solution:

Given information:

• Initial velocity: \(u = 2\) m/s

• Final velocity: \(v = 10\) m/s

• Displacement: \(s = 48\) m

• Time: \(t = ?\) (to find in part a)

• Acceleration: \(a = ?\) (to find in part b)

Part a) Find the time taken

Using \(s = \frac{1}{2}(u+v)t\):

\[48 = \frac{1}{2}(2 + 10)t\]

\[48 = \frac{1}{2}(12)t\]

\[48 = 6t\]

\[t = 8\text{ s}\]

Answer to part a):

The time taken for the travel is 8 seconds

Part b) Find the acceleration

Using \(v = u + at\):

\[10 = 2 + a(8)\]

\[10 = 2 + 8a\]

\[8 = 8a\]

\[a = 1\text{ m/s}^2\]

Answer to part b):

The acceleration of the object is 1 m/s²

Verification:

We can verify using \(v^2 = u^2 + 2as\):

\[10^2 = 2^2 + 2(1)(48)\]

\[100 = 4 + 96\]

\[100 = 100\] ✓

Key insight:

When you know initial velocity, final velocity, and displacement, using \(s = \frac{1}{2}(u+v)t\) is often the most efficient way to find time, as it's a simple linear equation.

Time taken to reach various points along its path

\[s = ut + \frac{1}{2}at^2\]

A particle is projected vertically upwards with a speed 19.6 ms\(^{-1}\). a) Calculate the time taken for the particle to: i) reach 14.7m above its starting point ii) be at 14.7m above its starting point for the second time. b) How long will the particle be above 14.7m?

Learn More →

Solution:

Given information:

• Initial velocity: \(u = 19.6\) m/s (upwards, positive)

• Acceleration: \(a = -9.8\) m/s² (gravity, downwards, negative)

• Displacement: \(s = 14.7\) m

• Time: \(t = ?\) (what we need to find)

Part a) i) Time to reach 14.7m (first time)

Using \(s = ut + \frac{1}{2}at^2\):

\[14.7 = 19.6t + \frac{1}{2}(-9.8)t^2\]

\[14.7 = 19.6t - 4.9t^2\]

Rearranging:

\[4.9t^2 - 19.6t + 14.7 = 0\]

Dividing by 4.9:

\[t^2 - 4t + 3 = 0\]

Factoring:

\[(t - 1)(t - 3) = 0\]

Solutions: \(t = 1\) s or \(t = 3\) s

Answer to part a) i):

The particle reaches 14.7m for the first time at \(t = 1\) s (on the way up)

Part a) ii) Time at 14.7m (second time)

Answer to part a) ii):

The particle is at 14.7m for the second time at \(t = 3\) s (on the way down)

Part b) Time above 14.7m

The particle is above 14.7m between \(t = 1\) s and \(t = 3\) s

Time above 14.7m = \(3 - 1 = 2\) s

Answer to part b):

The particle will be above 14.7m for 2 seconds

Physical interpretation:

The particle passes through 14.7m twice - once while ascending at \(t = 1\) s and once while descending at \(t = 3\) s. This symmetry occurs because the motion is uniformly accelerated.

Speed of projection and maximum height reached

\[v = u + at\]

An object is thrown vertically upwards. It takes 7s to return to its original position. Taking g = 9.8 ms\(^{-2}\), calculate a) the speed of projection (initial speed) b) the maximum height reached by the object.

Learn More →

Solution:

Given information:

• Total time to return to starting position: \(t = 7\) s

• Acceleration: \(a = -9.8\) m/s² (gravity, downwards, negative)

• Displacement when returned: \(s = 0\) m

• Initial velocity: \(u = ?\) (what we need to find)

Part a) Find the speed of projection

Using \(s = ut + \frac{1}{2}at^2\):

\[0 = u(7) + \frac{1}{2}(-9.8)(7)^2\]

\[0 = 7u + \frac{1}{2}(-9.8)(49)\]

\[0 = 7u - 240.1\]

\[7u = 240.1\]

\[u = 34.3\text{ m/s}\]

Answer to part a):

The speed of projection is 34.3 m/s

Part b) Find the maximum height

At maximum height, the velocity is zero: \(v = 0\)

Using \(v^2 = u^2 + 2as\):

\[0^2 = (34.3)^2 + 2(-9.8)s\]

\[0 = 1176.49 - 19.6s\]

\[19.6s = 1176.49\]

\[s = 60.025 \approx 60.0\text{ m}\]

Answer to part b):

The maximum height reached is 60.0 m

Alternative method for part b):

Since the motion is symmetric, the time to reach maximum height is half the total time:

\[t_{max} = \frac{7}{2} = 3.5\text{ s}\]

Using \(s = ut + \frac{1}{2}at^2\):

\[s = 34.3(3.5) + \frac{1}{2}(-9.8)(3.5)^2\]

\[s = 120.05 - 60.025\]

\[s = 60.025 \approx 60.0\text{ m}\]

Physical interpretation:

The object takes 3.5s to reach maximum height and 3.5s to fall back down, demonstrating the symmetry of vertical motion under constant acceleration.
